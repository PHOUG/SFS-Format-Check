## Описание файлов:

### ```format.c``` - форматирование SFS:
1. Создаёт файл ```disk.img``` размером 1 МБ

2. Записывает суперблок

3. Инициализирует битовую карту (```bitmap```)

4. Создаёт таблицу ```inodes```

5. Оставляет место под блоки данных
